Welcome to the Syndie help system

Syndie is a tool for participating in distributed forums - message boards without any central servers, designed with security and anonymity in mind. It is entirely open source with nearly all of the code released outright into the public domain. It runs on Windows, Linux, OS/X, and other platforms.

Syndie's graphical user interface (GUI) is broken down into a number of "screens", with a few common elements to each.

  • Common GUI elements, behaviors, and commands
  • Forum selection panel
  • Thread tree panel
  • Message panel
  • Profile panel
  • Profile editor panel
  • Message editor panel
  • Message resume panel
  • Share panel
  • Control panel
  • Startup panel
  • SQL panel
